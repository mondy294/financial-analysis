# 基金走势分析参考手册

用于基金分析 Agent 在遇到特定场景时参考，不是死规则。

## 1. 推荐输出顺序

### 1.1 先写最近一周发生了什么
优先描述：
- 最近一周是连续上行、冲高回落、低位反弹、弱势横盘还是继续下探
- 最近一周变化是否改变了 MA5 / MA10 / MA20 的相对位置
- 这一周是趋势延续还是短期扰动

更稳妥的表述：
- “最近一周更像是……，目前仍处在……结构里。”
- “这周的反弹更接近……，还不能直接当作趋势反转。”

### 1.2 再写可能原因
优先从这些角度找原因：
- 同类基金相对位置变化
- 重仓股普涨普跌或集中拉动
- 近期回撤后的技术修复
- 均线修复、乖离收敛或均值回归
- 高波动基金的短期弹性

不要写成确定事实，建议表述：
- “可能原因之一是……”
- “从持仓股广度和同类表现看，更像是……”
- “目前没有足够证据证明是……，更像是短期……”

## 2. 常见结论映射

### 2.1 趋势偏多
常见组合：
- 最新净值站上 MA20 / MA60
- MA5 >= MA10 >= MA20
- 近 20 / 60 日收益为正
- 同类排名不差，且回撤没有明显失控

更合适的建议表述：
- 已持有：持有待跟踪，若回撤到 MA10-MA20 区间且证据未恶化，可考虑加仓 10%-15%
- 未持有：分批布局，先用计划投入资金的 10%-15% 试探，不追单日大涨

### 2.2 中性震荡
常见组合：
- MA5、MA10、MA20 反复缠绕
- 近 1 月收益接近 0
- 同类排名居中
- 波动与回撤都不算极端

更合适的建议表述：
- 已持有：继续观察，维持现有仓位，暂不主动加仓；若后续站稳 MA20 再考虑补 5%-10%
- 未持有：先观察，等趋势更明确；若必须参与，只做 5%-10% 的试探仓

### 2.3 偏谨慎
常见组合：
- 最新净值跌破 MA20 / MA60
- 近 60 / 120 日收益转弱
- 回撤和波动在同类里偏差
- 重仓股近期明显走弱，且集中度较高

更合适的建议表述：
- 已持有：谨慎减仓 10%-20%，先降低波动暴露，避免把反弹想成反转
- 未持有：观望为主，先别抢反弹

### 2.4 无法判断
常见组合：
- 数据不足
- 基金类型和当前分析框架明显不匹配
- 趋势、同类位置、持仓广度互相冲突

更合适的建议表述：
- 已持有：先不做大动作，维持观察，等待更多确认信号
- 未持有：暂缓介入

## 3. 基金类型修正

### 主动权益 / 偏股混合
- 对趋势、回撤、持仓股广度更敏感
- 操作建议可以更强调节奏和仓位管理

### 指数基金 / ETF 联接
- 要多看赛道和行业暴露
- 如果只是板块短期过热，建议中要提醒回撤风险

### 纯债 / 固收+
- 不应沿用高弹性基金的话术
- 如果波动和回撤异常，反而要重点提示“稳定性下降”
- 仓位动作应更保守，通常不建议大幅加减

### QDII
- 注意海外市场、汇率、时差等影响
- 即使短期表现较强，也要保留更高的不确定性提示

### FOF
- 节奏通常慢于高弹性权益基金
- 评价时更看稳健性与回撤控制

## 4. 风险提示优先级
优先写最影响操作的风险：
1. 回撤继续扩大
2. 重仓股集中且短期转弱
3. 同类排名下滑
4. 估值/净值偏离过大后的均值回归
5. 海外、汇率或单一主题暴露

## 5. 仓位动作的边界
- 可以给出明确动作，但必须使用保守范围，而不是绝对化数字。
- 更推荐：
  - “加仓 10%-15% 当前持仓金额”
  - “减仓 10%-20%”
  - “维持现有仓位，暂不新增”
  - “先用计划投入资金的 10%-15% 试探建仓”
- 不推荐：
  - “建议满仓”
  - “直接加仓 50% 以上”
  - “一定会涨，可以重仓”

## 6. 不要这么写
- “一定会上涨”
- “现在就是最佳买点”
- “建议满仓”
- “短期必然反弹”
- “只看最近一天涨跌就给方向”
- “因为市场情绪很好所以会涨”这类没有数据支撑的话

## 7. 更稳妥的表述模板
- “最近一周更像是……，可能原因包括……，但目前更像……而不是已经确认反转。”
- “从趋势和同类位置看，未来 1-3 个月更偏向……，但需要留意……”
- “当前更适合……，核心依据是……；如果后续出现……则需要重新评估。”
- “如果按当前持仓处理，更建议……，动作幅度控制在……会更稳妥。”
- “这更像一只……角色的基金，不适合用……的方式去操作。”
