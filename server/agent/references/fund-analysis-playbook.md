# 基金走势分析参考手册

用于基金分析 Agent 在遇到特定场景时参考，不是死规则。

## 1. 常见结论映射

### 1.1 趋势偏多
常见组合：
- 最新净值站上 MA20 / MA60
- MA5 >= MA10 >= MA20
- 近 20 / 60 日收益为正
- 同类排名不差，且回撤没有明显失控

更合适的建议表述：
- 已持有：持有待跟踪、逢回撤再看是否加仓
- 未持有：分批布局，不追单日大涨

### 1.2 中性震荡
常见组合：
- MA5、MA10、MA20 反复缠绕
- 近 1 月收益接近 0
- 同类排名居中
- 波动与回撤都不算极端

更合适的建议表述：
- 已持有：继续观察，不急着大幅调仓
- 未持有：先观察，等趋势更明确

### 1.3 偏谨慎
常见组合：
- 最新净值跌破 MA20 / MA60
- 近 60 / 120 日收益转弱
- 回撤和波动在同类里偏差
- 重仓股近期明显走弱，且集中度较高

更合适的建议表述：
- 已持有：谨慎减仓或降低预期，避免把反弹想成反转
- 未持有：观望为主，先别抢反弹

## 2. 基金类型修正

### 主动权益/偏股混合
- 对趋势、回撤、持仓股广度更敏感
- 操作建议可以更强调节奏和仓位管理

### 指数基金 / ETF 联接
- 要多看赛道和行业暴露
- 如果只是板块短期过热，建议中要提醒回撤风险

### 纯债 / 固收+
- 不应沿用高弹性基金的话术
- 如果波动和回撤异常，反而要重点提示“稳定性下降”

### QDII
- 注意海外市场、汇率、时差等影响
- 即使短期表现较强，也要保留更高的不确定性提示

### FOF
- 节奏通常慢于高弹性权益基金
- 评价时更看稳健性与回撤控制

## 3. 风险提示优先级
优先写最影响操作的风险：
1. 回撤继续扩大
2. 重仓股集中且短期转弱
3. 同类排名下滑
4. 估值/净值偏离过大后的均值回归
5. 海外、汇率或单一主题暴露

## 4. 不要这么写
- “一定会上涨”
- “现在就是最佳买点”
- “建议满仓”
- “短期必然反弹”
- “只看最近一天涨跌就给方向”

## 5. 更稳妥的表述模板
- “从趋势和同类位置看，未来 1-3 个月更偏向……，但需要留意……”
- “当前更适合……，核心依据是……，如果后续出现……则需要重新评估。”
- “这更像一只……角色的基金，不适合用……的方式去操作。”
